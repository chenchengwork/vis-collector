### 制作sprite的过程

1. 上传待制作的icon并可以编辑icon
```
通过icon-editor

https://www.mapbox.com/maki-icons/editor/

https://github.com/mapbox/maki

```

2. 通过spritezero生成sprite@2x.png和sprite@2x.json
```
https://github.com/mapbox/spritezero

```

3. 制作过程
```
(1). 将所有的svg文件都放入input目录中

(2). 输出结果放在output目录中

(3). 执行./index.js

```
