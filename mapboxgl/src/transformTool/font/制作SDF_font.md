### 制作font的过程

https://github.com/mapbox/node-fontnik

1. 制作原理
```

一次性加载中文字体库会遇到文件过大，加载时间过长的问题。Mapbox GL JS借鉴了地图切片的原理，根据字符编码范围进行分段，将[0, 65535]的字符按照每段长度256的方式分为若干段，每段以start-end.pbf的格式进行命名。在加载时，根据字符的编码确定该字符所在的分段，然后请求相关的文件

```

2. 制作过程
```
(1). 将字体文件都放入input目录中

(2). 输出结果放在output目录中

(3). 执行./index.js

```
